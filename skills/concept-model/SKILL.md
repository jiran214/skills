# 概念模型生成

## Workflow

1. 理解用户输入的概念
2. 识别核心因素（3-7个）
3. 确定因素间关系（加减、乘除、正负相关、结构关系）
4. 输出模型（纯文本，不超过10行）

## 示例

用户：/concept-model 学习效率

```
学习效率 = 注意力 × 方法质量 / 时间浪费

解释：
- 注意力：专注程度，走神越多效率越低
- 方法质量：学习策略的有效性
- 时间浪费：无效学习时间占比

要点：提升效率的关键是减少浪费、保持专注、优化方法。
```
