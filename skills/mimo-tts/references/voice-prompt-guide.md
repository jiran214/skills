# 音色描述 Voice Design Prompt 指南

## 核心原则

- 1-4句即可，核心特征比堆砌维度更重要
- 中英文均可
- 避免矛盾特征（如"低沉又尖锐"）
- 避免音质效果词（混响、回声等，这些是后期处理）
- 避免过于模糊的词（如"好听的声音"）

## 描述维度

### 必选维度

| 维度 | 中文示例 | 英文示例 |
|------|----------|----------|
| 性别与年龄 | "二十多岁的年轻女性" | "a young woman in her mid-20s" |
| 音色/质感 | "丝滑醇厚、带着磁性" | "deep and gravelly" |

### 可选维度

| 维度 | 中文示例 | 英文示例 |
|------|----------|----------|
| 情绪/语气 | "温柔但带着一丝疲惫" | "warm and confident" |
| 语速/节奏 | "语速极快，像连珠炮" | "slow and deliberate" |
| 角色/人设 | "像新闻联播的播音员" | "like a nature documentary narrator" |
| 说话风格 | "咬字清晰，字正腔圆" | "casual and laid-back" |
| 场景描写 | "像在你耳边轻声说话" | "as if telling a bedtime story" |
| 年代参照 | "八十年代广播电台的感觉" | "vintage 1940s radio announcer" |

## 两种写法风格

### 简洁描述型

用关键词快速勾勒声音轮廓，适合大多数场景：

```
年轻女性，声音清澈甜美，语速适中
```

```
A warm, middle-aged male voice with a slight British accent.
```

### 专业描述型

通过场景、人设或多维度细节立体刻画，适合对音色有精细要求：

```
一位四十岁左右的女性，声音低沉富有磁性，说话不急不缓，像是深夜电台的情感节目主持人，带着一种让人安心的温暖感。
```

```
A young man in his early 20s with a bright, energetic voice. He speaks fast with lots of enthusiasm, like a tech YouTuber doing an unboxing video. Slightly informal, uses contractions naturally.
```

## 常见场景示例

### 有声书朗读
```
成熟女性，声音温柔沉稳，语速偏慢，像在给孩子讲睡前故事
```

### 新闻播报
```
中年男性，字正腔圆，声音浑厚有力，播音腔，语速适中偏快
```

### 产品介绍
```
年轻女性，声音活泼明快，充满热情，语速稍快，像在向朋友推荐好东西
```

### 商务汇报
```
三十多岁男性，声音沉稳专业，咬字清晰，语速稳定，有说服力
```

### 解说旁白
```
A calm, authoritative male voice, like a seasoned nature documentary narrator. Measured pace, slightly dramatic emphasis on key points.
```

### 角色配音
```
一个调皮的小男孩，声音清脆响亮，说话带着搞怪的语气，时不时会笑出来
```

## 进阶技巧

1. **用比喻比直接描述更有效**："像深夜电台主持人"比"低沉温暖的男声"更能传达氛围
2. **情绪叠加可以产生层次**："温柔但带着一丝疲惫"比单纯的"温柔"更生动
3. **场景暗示语速和节奏**："像在耳边轻声说话"自然暗示了慢速和亲密感
4. **合成文本应与音色匹配**：音色描述为"愤怒"但文本内容是平静的叙述，效果会不协调
